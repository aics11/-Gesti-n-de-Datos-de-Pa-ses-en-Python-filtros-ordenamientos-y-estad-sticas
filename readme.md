## Descripción

Sistema de gestión de información sobre países desarrollado en Python.
Permite agregar, buscar, filtrar, ordenar países y obtener estadísticas
a partir de un dataset cargado desde un archivo CSV.

## Cómo ejecutar

1. Clonar o descargar el repositorio
2. Asegurarse de que `paises.csv` esté en la misma carpeta que `paises.py`
3. Ejecutar en la terminal:

paises.py

---

## Funcionalidades

- **Agregar país**: ingresa un nuevo país con nombre, población, superficie y continente
- **Actualizar**: modifica población y superficie de un país existente
- **Buscar**: búsqueda por nombre con coincidencia parcial o exacta
- **Filtrar**: por continente, rango de población o rango de superficie
- **Ordenar**: por nombre, población o superficie (ascendente o descendente)
- **Estadísticas**: país con mayor/menor población, promedios y cantidad por continente
